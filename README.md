## Features

- **Stages** — 9 named workspaces spanning all monitors. Switch instantly with Opt+1-9.
- **Snap Zones** — 14 zones for halves, thirds, quarters, and portrait-adaptive layouts. Drag or use keyboard shortcuts.
- **Bento Box Tiling** — Automatic BSP tiling with iOS-style expel/swap reordering.
- **Snap Wheel** — Radial zone picker via Ctrl+Opt or middle-click.
- **Multi-Monitor** — Full support with display exclusion and portrait monitor awareness.
- **Keyboard Shortcuts** — Fully configurable with conflict detection.
